## The Production Workflow

### Step 1: Client Onboarding (5 minutes)

**Data Collection Form:**
```
1. What's your trade? [Dropdown: Plumber, Electrician, Retailer, etc.]
2. How many people in your team? [Number]
3. How many jobs per week? [Number]
4. Average job value? [£]
5. What's your biggest frustration? [Text]
6. What's your target annual income? [£]
7. How many weeks off do you want? [Number]
8. What do you want this business to enable? [Text]
```

**Auto-populates:**
- Script template
- Video template
- Calculations (missed revenue, potential uplift)

### Step 2: Template Selection (1 minute)

**Based on trade type:**
- Plumber → Plumbing template
- Electrician → Electrical template
- Retailer → Retail template
- etc.

**Based on pain point:**
- Missed calls → Lead capture template
- Scheduling → Efficiency template
- Marketing → Growth template

### Step 3: Script Generation (5 minutes)

**Auto-filled from data:**
```
"You're a [TRADE] with [TEAM] people, doing [JOBS] jobs a week.
You're missing [CALCULATED] calls a day. That's [CALCULATED] a week.
At a 50% conversion rate and your £[VALUE] average job value,
that's £[CALCULATED]/month. Gone."
```

**Review and adjust:**
- Check numbers are correct
- Adjust tone if needed
- Add client-specific details

### Step 4: Avatar/Voice Selection (2 minutes)

**Option A: Stock Avatar**
- Browse HeyGen library
- Filter by: age, gender, style (business casual/professional)
- Pick one that matches client's demographic

**Option B: Custom Avatar**
- Client records 2-minute video
- Upload to HeyGen
- Generate avatar (2-5 hours processing)
- Use for all their videos

**Option C: Voice Clone Only**
- Client records 1-minute voice sample
- HeyGen clones voice
- Use with stock avatar

### Step 5: B-Roll Generation (30 minutes)

**In Runway:**
- Generate 3-4 scenes using filled prompts
- Download best versions
- Organize by scene number

**Scenes to generate:**
1. Hook (phone screen, missed call)
2. Problem (cluttered desk, chaos)
3. Data (animated counter)
4. Solution (clean dashboard)
5. Amplification (growth graph)

### Step 6: HeyGen Generation (10 minutes)

**Upload script → Select avatar → Generate**
- Download presenter video
- 1080p quality

### Step 7: Editing in Descript (60 minutes)

**Stitch together:**
1. Import HeyGen video
2. Import Runway B-roll clips
3. Arrange on timeline
4. Add transitions (simple cuts)
5. Add captions (auto-generated)
6. Add music (from Epidemic Sound)
7. Add sound effects
8. Review pacing
9. Export final video

### Step 8: Delivery (5 minutes)

**Upload to client portal:**
- 1080p MP4 file
- Caption file (SRT)
- Thumbnail image
- Usage instructions (where to post, how to use)

**Send notification:**
"Your video is ready! Here's how to use it..."

## Risk Mitigation

### What Could Go Wrong?

**1. AI Tool Price Increases**
- Mitigation: Lock in annual plans
- Alternative: Build relationships, negotiate enterprise pricing
- Backup: Multiple tool options (HeyGen, Synthesia, etc.)

**2. Quality Inconsistency**
- Mitigation: Strict templates, checklists
- Quality control: Review every video before delivery
- Client feedback loop: Iterate based on feedback

**3. Client Expectations Too High**
- Mitigation: Clear scope, show examples upfront
- Set expectations: "Professional, not Hollywood blockbuster"
- Revisions policy: 2 rounds included, then charge

**4. Tool Downtime/Outage**
- Mitigation: Multiple tool accounts
- Backup: Pre-rendered template elements
- Communication: Proactive updates to clients
