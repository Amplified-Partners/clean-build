# Kilo Code Profiles & Provider Strategy

This document defines the optimized profiles for different development tasks within Kilo Code.

## 1. Local-Dev-Coding
**Goal:** Fast, private coding loops for rapid iteration.
**Provider:** Local (Ollama / LM Studio)
**Model:** `qwen2.5-coder:7b` (or `deepseek-coder:6.7b` if available)
**Settings:**
- Temperature: 0.1 (Precise)
- Max Tokens: 2048
- Context Window: 8192 (Keep it light for speed)
**Context Strategy:**
- Focus on current file + immediate imports.
- Avoid indexing `node_modules` or build artifacts.
**Guardrails:**
- Auto-apply small diffs.
- Require approval for file deletion or shell commands.

## 2. Cloud-Deep-Refactor
**Goal:** Large-scale refactoring, architectural changes, and complex logic.
**Provider:** Anthropic
**Model:** `claude-3-5-sonnet-20240620` (or latest stable)
**Settings:**
- Temperature: 0.3 (Balanced)
- Max Tokens: 8192
- Context Window: 200k (Leverage full context)
**Context Strategy:**
- Index full `src/` directory.
- Prioritize `README.md`, `API.md`, and core type definitions.
**Guardrails:**
- Require approval for all file writes.
- Show comprehensive diffs before applying.

## 3. Railway-Ops-Agent
**Goal:** Debugging, tuning, and managing Railway deployments.
**Provider:** Anthropic (for reasoning) or OpenAI (for tool use)
**Model:** `claude-3-5-sonnet` or `gpt-4o`
**Settings:**
- Temperature: 0.1
- System Prompt Supplement: "You are a DevOps expert specialized in Railway, Docker, and PostgreSQL. Focus on logs, metrics, and configuration."
**Context Strategy:**
- Index `railway.toml`, `Dockerfile`, `docker-compose.yml`.
- Access to `docs/railway-runbooks.md`.
**Guardrails:**
- **CRITICAL:** Never execute production database writes without explicit confirmation.
- Read-only access to logs/metrics by default.

## 4. Docs-Research-Agent
**Goal:** Reading documentation, generating runbooks, and commenting code.
**Provider:** Anthropic (Haiku) or Local (Llama 3 8B)
**Model:** `claude-3-haiku` or `llama3`
**Settings:**
- Temperature: 0.5 (Creative/Explanatory)
- Max Tokens: 4096
**Context Strategy:**
- Index `docs/` folder.
- Focus on markdown files and comments.
