# covered.AI Logo Concepts

## Executive Summary
Three distinct logo concepts for covered.AI, each communicating different aspects of the brand: background optimization, protective support, and systematic tightening. All concepts prioritize low-friction positioning and UK SMB accessibility.

---

## Concept 1: Background Layer

### Visual Description
A layered icon combining three overlapping translucent shapes in a triangular composition. Each layer represents a different business function (operations, cash flow, admin) working together. The layers create depth and movement without disruption. The wordmark "covered.AI" sits below the icon with the period after "covered" emphasizing the .AI positioning.

### Icon Style
- Geometric abstract shapes with 20% opacity fill
- Soft edges, no harsh lines
- Dynamic but calm composition
- SVG vector format for scalability

### Typography Pairing
- Primary: Satoshi Bold (brand name)
- Secondary: Inter Regular (tagline)
- Monospace: JetBrains Mono for .AI accent

### Color Application
- Primary Navy (#0A2540): Main brand color for text and key elements
- Secondary Teal (#00D9C0): Accent color for the topmost layer
- White space: Negative space between layers
- Light Gray (#F7FAFC): Background for light mode contexts

### Use Cases
- Website header and favicon
- Email signatures and business cards
- Social media profile pictures
- Invoice and document headers

### Why It Works
- The layered approach visually represents "working in the background" without demanding attention
- Transparency suggests visibility into operations without disruption
- Triangular composition implies stability and support
- Simple enough for favicon (32px), bold enough for billboard

### AI Image Generation Prompt
```
Modern minimalist logo for UK business automation company covered.AI. Three overlapping translucent geometric layers in triangular composition. Top layer in bright teal (#00D9C0), middle layer in navy blue (#0A2540), bottom layer in soft gray. Semi-transparent fills showing layer depth. Clean, professional, tech-forward but not flashy. ## Concept 2: Safety Net

### Visual Description
A stylized network node structure that resembles both a protective umbrella and a connected system. The lines create a subtle shield shape while maintaining the interconnected look of automated systems. Small connection points along the lines suggest automation and communication between business functions.

### Icon Style
- Line art with consistent stroke weight (2px)
- Geometric but approachable
- Connection nodes as small filled circles
- Clean, scalable vector format

### Typography Pairing
- Primary: Satoshi Bold (brand name)
- Secondary: Inter Regular (tagline)
- Monospace: JetBrains Mono for .AI accent

### Color Application
- Primary Navy (#0A2540): Main stroke color
- Secondary Teal (#00D9C0): Connection nodes and accent points
- Charcoal Gray (#2D3748): Body text applications
- White: Stroke fill

### Use Cases
- LinkedIn company page avatar
- Website favicon and browser tab icon
- Invoice PDF headers
- Social media profile images

### Why It Works
- Network node metaphor communicates systems working together
- Shield shape subtly suggests "covered" and protection
- Connection points represent automation and communication
- Works well at small sizes (favicon, social avatars)

### AI Image Generation Prompt
```

Connection nodes as small bright teal (#00D9C0) dots along the lines. Navy blue (#0A2540) stroke color. Professional, trustworthy, not flashy. White background. Company name "covered.AI" below in bold modern sans-serif font. British aesthetic, understated confidence. Vector-style flat design, no gradients, no shadows.

## Concept 3: Tightening Mechanism

### Visual Description
A stylized gear or tightening symbol that suggests optimization without industrial heaviness. The gear has soft rounded edges and incorporates a subtle checkmark or flow line through the center, representing the "tightening" of processes. The overall shape suggests efficiency and systematic improvement.

### Icon Style
- Simplified gear shape with rounded corners
- Soft, approachable geometry
- Single accent line or flow element
- Clean vector format

### Typography Pairing
- Primary: Satoshi Bold (brand name)
- Secondary: Inter Regular (tagline)
- Monospace: JetBrains Mono for .AI accent

### Color Application
- Primary Navy (#0A2540): Main gear shape
- Secondary Teal (#00D9C0): Flow/optimization accent line
- Light Gray (#F7FAFC): Background contexts
- Charcoal (#2D3748): Text applications

### Use Cases
- Website footer and brand materials
- Proposal and document headers
- Presentation slides
- Merchandise and branding items

### Why It Works
- Gear metaphor directly communicates "tightening" and optimization
- Simplified design avoids industrial/manufacturing associations
- Flow line suggests improvement without transformation
- Distinctive shape aids brand recall

## File Specifications

### Primary Logo (Full Lockup)
- **Format**: SVG, PNG
- **Sizes**: 512x512, 1024x1024, 2048x2048 pixels
- **Clear Space**: 20px minimum on all sides
- **Orientation**: Horizontal lockup (icon left, text right)

### Icon-Only Version
- **Format**: SVG, PNG
- **Sizes**: 32x32 (favicon), 64x64, 128x128, 256x256 pixels
- **Use Cases**: Favicon, app icon, social media avatars

### Wordmark-Only Version
- **Format**: SVG, PNG
- **Sizes**: Horizontal formats only
- **Use Cases**: Email signatures, document headers, merchandise

### Reversed Version (Dark Backgrounds)
- **Format**: SVG, PNG
- **Colors**: White icon and text on dark backgrounds
- **Use Cases**: Dark mode website, presentations, video overlays

### Favicon
- **Format**: ICO, PNG
- **Sizes**: 16x16, 32x32, 48x48 pixels
- **Recommendation**: Use icon-only version at 32x32

### PDF Version
- **Format**: PDF (vector-based)
- **Use Cases**: Print materials, professional documents, contracts

## Brand Lock-up Rules

### Horizontal Lockup (Primary)
```
[ICON - 48px] [SPACE - 16px] [covered.AI - 32px]
```

### Stacked Lockup (Alternative)
```
      [ICON - 64px]
      
[covered.AI - 28px]
```

### Clear Space Requirements
- Minimum clear space = height of the icon's "O" character
- Never挤压 logo elements together
- Maintain consistent spacing in all applications

### Minimum Sizes
- Digital (screen): 32px width minimum
- Print: 0.5 inches / 12mm width minimum
- Never scale below these thresholds

### Never-Do List
- ❌ Do not stretch or distort the logo
- ❌ Do not rotate the logo
- ❌ Do not add drop shadows or effects
- ❌ Do not change the colors (except reversed version)
- ❌ Do not place on busy or low-contrast backgrounds
- ❌ Do not separate the icon from the wordmark without purpose
- ❌ Do not use gradients on any logo elements
- ❌ Do not add decorative elements or frames

## Recommended Concept

**Concept 1: Background Layer** is recommended as the primary logo for covered.AI.

**Rationale:**
1. Most directly communicates the "background operation" positioning
2. Transparency metaphor aligns with low-friction, no-disruption messaging
3. Works well at all sizes (favicon to billboard)
4. Distinctive enough for brand recall without being flashy
5. Complements rather than competes with content

**Secondary Recommendation:**
Use **Concept 2: Safety Net** for social media avatars and favicon applications where the simplified network structure reads better at small sizes.

## Designer Handoff Notes

When working with a designer to finalize:

1. Provide this document as the primary reference
2. Share the hex codes: Navy #0A2540, Teal #00D9C0, Charcoal #2D3748, Light Gray #F7FAFC
3. Specify font requirements: Satoshi Bold, Inter Regular, JetBrains Mono
4. Request SVG format for all vector work
5. Request both light and reversed (dark) versions
6. Ask for favicon.ico generation from the icon-only version
7. Request usage examples in context (website header, business card, email signature)
