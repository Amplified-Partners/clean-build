# Financial System — Master Specification
## Complete Invoice-to-Cash Pipeline

---

## VISION

**From:** Create invoice → Send email → Chase manually → Mark paid manually → Tell accountant

**To:** Create invoice → Send with payment link → Auto-chase → Payment received → Auto-reconciled → Synced to Xero → Accountant happy

## DATA MODEL ADDITIONS

```prisma
// Add to existing Invoice model
model Invoice {
  // ... existing fields
  
  // PDF & Document
  pdfUrl              String?
  pdfGeneratedAt      DateTime?
  
  // Stripe
  stripeInvoiceId     String?
  stripePaymentIntentId String?
  paymentLinkUrl      String?
  paymentLinkExpiresAt DateTime?
  
  // Xero
  xeroInvoiceId       String?
  xeroSyncedAt        DateTime?
  xeroSyncError       String?
  
  // Reminders
  reminder1SentAt     DateTime?
  reminder2SentAt     DateTime?
  reminder3SentAt     DateTime?
  finalNoticeSentAt   DateTime?
  
  // Payment
  paidAt              DateTime?
  paidAmount          Decimal?
  paymentMethod       String?    // stripe, bank_transfer, cash, cheque
  paymentReference    String?
  stripeChargeId      String?
  
  // Tracking
  emailOpenedAt       DateTime?
  linkClickedAt       DateTime?
  viewedAt            DateTime?
}

// Add to existing Quote model
model Quote {
  // ... existing fields
  
  // PDF & Document
  pdfUrl              String?
  pdfGeneratedAt      DateTime?
  
  // Tracking
  emailOpenedAt       DateTime?
  linkClickedAt       DateTime?
}

// New: Stripe Connect accounts
model StripeAccount {
  id                  String   @id @default(cuid())
  clientId            String   @unique
  client              Client   @relation(fields: [clientId], references: [id])
  
  stripeAccountId     String   @unique  // acct_xxx
  chargesEnabled      Boolean  @default(false)
  payoutsEnabled      Boolean  @default(false)
  onboardingComplete  Boolean  @default(false)
  
  // Account details
  businessName        String?
  email               String?
  country             String   @default("GB")
  defaultCurrency     String   @default("gbp")
  
  // Payout schedule
  payoutSchedule      String   @default("daily")  // daily, weekly, monthly
  
  createdAt           DateTime @default(now())
  updatedAt           DateTime @updatedAt
}

// New: Xero connection
model XeroConnection {
  id                  String   @id @default(cuid())
  clientId            String   @unique
  client              Client   @relation(fields: [clientId], references: [id])
  
  tenantId            String   // Xero org ID
  tenantName          String?
  
  // OAuth tokens (encrypted)
  accessToken         String
  refreshToken        String
  tokenExpiresAt      DateTime
  
  // Sync settings
  autoSyncInvoices    Boolean  @default(true)
  autoSyncPayments    Boolean  @default(true)
  defaultAccountCode  String?  // Revenue account
  
  // Status
  connected           Boolean  @default(true)
  lastSyncAt          DateTime?
  lastSyncError       String?
  
  createdAt           DateTime @default(now())
  updatedAt           DateTime @updatedAt
}

// New: Payment events log
model PaymentEvent {
  id                  String   @id @default(cuid())
  invoiceId           String
  invoice             Invoice  @relation(fields: [invoiceId], references: [id])
  
  eventType           String   // payment_created, payment_succeeded, payment_failed, refund
  eventData           Json
  
  stripeEventId       String?  @unique
  
  createdAt           DateTime @default(now())
}
```

## API ROUTES OVERVIEW

### Invoice Documents
```
POST   /api/v1/invoices/{id}/generate-pdf
GET    /api/v1/invoices/{id}/pdf
POST   /api/v1/invoices/{id}/send
GET    /api/v1/invoices/{id}/view/{token}  (public)
```

### Stripe Payments
```
POST   /api/v1/stripe/connect/create-account
GET    /api/v1/stripe/connect/onboarding-link
GET    /api/v1/stripe/connect/status
POST   /api/v1/invoices/{id}/create-payment-link
POST   /api/v1/webhooks/stripe  (public)
```

### Auto-Chase
```
GET    /api/v1/invoices/overdue
POST   /api/v1/invoices/{id}/send-reminder
POST   /api/v1/invoices/{id}/send-final-notice
```

### Xero Integration
```
GET    /api/v1/xero/connect
GET    /api/v1/xero/callback
GET    /api/v1/xero/status
POST   /api/v1/xero/disconnect
POST   /api/v1/invoices/{id}/sync-to-xero
POST   /api/v1/xero/sync-all
```

## TRIGGER.DEV JOBS

| Job | Schedule | Description |
|-----|----------|-------------|
| `generate-invoice-pdf` | On demand | Generate PDF for invoice |
| `send-invoice` | On demand | Send invoice email with PDF |
| `send-invoice-reminder` | On demand | Send reminder email |
| `check-overdue-invoices` | Daily 9am | Find overdue, trigger reminders |
| `sync-invoice-to-xero` | On demand | Push invoice to Xero |
| `refresh-xero-tokens` | Hourly | Refresh expiring OAuth tokens |
| `reconcile-stripe-payments` | Every 15 min | Check for new payments |
