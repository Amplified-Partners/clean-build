# Debug Issue

You are systematically debugging an issue. Follow these steps:

1. Use `ask_followup_question` to get error description or symptoms
2. Use `search_files` to find related code
3. Use `read_file` to examine suspect files
4. Identify probable cause
5. Propose fix and ask for confirmation before applying

Parameters needed (ask if not provided):
- Error message or symptom description
