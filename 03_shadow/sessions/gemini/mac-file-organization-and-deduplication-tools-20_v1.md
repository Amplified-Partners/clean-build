## Gemini

It is unlikely you are looking for "Anti-Gravity."

There is **no Mac software called "Anti-Gravity"** that organizes or deduplicates files. You may be thinking of the new developer tool from Google called *Antigravity* (which is for writing code, not organizing files), or you might be confusing the name with **Gemini 2**, which is currently the most popular app for file deduplication on Mac.

To achieve your goals of deduplicating, labeling, and organizing, here are the actual industry-standard tools you should use.

### 1. For Deduplicating Files
If you have multiple copies of photos, documents, or downloads, these apps find them and help you delete the extras safely.

* **Gemini 2 (Best Choice):** This is likely the app you heard about. It scans your Mac for duplicates and "similar" files (like 3 photos of the same sunset) and uses a smart algorithm to select the best one to keep.
    * *Cost:* Paid (approx. $20/year), but has a free trial.
* **dupeGuru:** A completely free, open-source alternative. It is less "pretty" than Gemini 2 but very powerful for finding duplicate music and documents.
    * *Cost:* Free.

### 2. For Organizing and Labeling
If you want files to "move themselves" into the right folders based on names or dates, you need automation software.

* **Hazel (The Gold Standard):** Hazel acts like a housekeeper for your Mac. You set simple rules (e.g., "If a file in Downloads is a PDF and has 'Invoice' in the name, move it to the 'Finances' folder and label it Red").
    * *Why it fits your request:* It runs silently in the background, keeping your folders organized without you touching them.
    * *Cost:* Paid ($42 one-time).

### 3. The Free "Built-in" Way (Finder)
You can do much of this without buying software using features already on your Mac.

* **Smart Folders:** You can create a "virtual" folder that instantly gathers files based on criteria.
    * *How to do it:* Open Finder > File > **New Smart Folder**.
    * *Example:* Create a smart folder that shows "All PDF files created in the last week."
* **Finder Tags:** You can right-click any file to add a colored "Tag" (Label). You can then click the tag in the Finder sidebar to see all files associated with that project, regardless of where they are stored.

### Summary Recommendation
If you want to clean up a messy Mac:
1.  Download **Gemini 2** to delete the duplicates.
2.  Use **Hazel** if you want to automate future organization (auto-filing).
3.  Use **Finder Tags** for manual labeling.
