### How AI Search Differs from Google

**Google ranks:** Individual pages based on keywords, backlinks, authority

- **Clear, direct answers** (not buried in paragraphs)

## PART 3: GOOGLE CORE WEB VITALS 2026 (Technical Excellence)

### The Three Metrics That Matter

**1. Largest Contentful Paint (LCP) - Loading Speed**
- **Target:** <2.5 seconds
- **What:** How fast does your main content load?
- **Impact:** Fast loading = more engagement, higher rankings, more conversions

**2. Interaction to Next Paint (INP) - Responsiveness**
- **Target:** <100ms
- **What:** How fast does the site respond to clicks?
- **Impact:** Smooth interaction = trust & engagement

**3. Cumulative Layout Shift (CLS) - Visual Stability**
- **Target:** <0.1
- **What:** Does content jump around as page loads?
- **Impact:** Stable layout = no accidental clicks, professional feel

### Your CWV Optimization Plan

**Priority 1: LCP (Biggest impact on rankings)**
- [ ] Use Webflow (handles optimization by default)
- [ ] Compress hero images (use WebP format)
- [ ] Lazy-load images below fold
- [ ] Use CDN for global edge delivery
- [ ] Minimize render-blocking JavaScript
- [ ] Preload critical resources
- [ ] Target: <2.0s (exceeds Google's requirement)

**Priority 2: INP (Growing importance in 2026)**
- [ ] Minimize main thread blocking
- [ ] Break up long JavaScript tasks
- [ ] Use event delegation for click handlers
- [ ] Optimize form interactions
- [ ] Target: <80ms (excellent)

**Priority 3: CLS (Consistency matters)**
- [ ] Reserve space for images (set width/height)
- [ ] Avoid pop-ups that shift content
- [ ] Load web fonts properly
- [ ] Avoid layout-triggering animations
- [ ] Target: <0.05 (perfect stability)

**Monthly monitoring:**
- Google Search Console (real user data)
- PageSpeed Insights (both desktop & mobile)
- Web Vitals Chrome extension
- Track metrics by page type (homepage, blog, landing pages)
