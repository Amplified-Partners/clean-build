**Format:** Interactive Web Calculator + Video + PDF

```python
# The True Hourly Rate Calculator

def calculate_true_hourly_rate():
    """
    What most SMB owners get wrong:
    They charge £50/hr but actually earn £18/hr
    """

    # INPUTS
    annual_revenue = input("Annual revenue (£)")
    direct_costs = input("Materials, subcontractors (£)")
    overheads = input("Rent, insurance, vehicles, tools (£)")
    your_salary = input("What you actually take (£)")
    hours_worked = input("Hours you work per week") * 48  # weeks

    # CALCULATIONS
    gross_profit = annual_revenue - direct_costs
    net_profit = gross_profit - overheads - your_salary
    true_hourly = (your_salary + net_profit) / hours_worked

    # OUTPUT
    return {
        "gross_margin": (gross_profit / annual_revenue) * 100,
        "net_margin": (net_profit / annual_revenue) * 100,
        "true_hourly_rate": true_hourly,
        "billable_hours_needed": target_income / true_hourly,
        "price_increase_needed": calculate_gap(true_hourly, target_hourly)
    }
```

- Calculate what they SHOULD charge

### Giveaway: "The 10-Minute SOP Method"

**Format:** Video + Template + Audio

**The Method:**

```markdown
# HOW TO DOCUMENT ANY PROCESS IN 10 MINUTES

You don't need fancy software. You need your phone.

## Step 1: Wait for the question (1 min)

Next time someone asks you "how do I...?"
DON'T just answer it.
Say: "Let me show you, and I'll record it so you've got it
for next time."

## Step 2: Screen record or film yourself (5 mins)

If it's on a computer:
- Start screen recording (Loom is free, or just use QuickTime)
- Talk through what you're doing and WHY
- Don't edit, don't restart - mistakes are learning moments

If it's physical:
- Prop up your phone
- Talk through each step
- Film your hands, not your face

## Step 3: Save it somewhere findable (2 mins)

Create a folder called "How We Do Things"
Put it in:
- Google Drive
- Dropbox
- Shared network drive
- Wherever your team can access it

Name the file clearly:
"How to process a refund"
"How to set up a new customer"
"How to close up at night"

## Step 4: Tell people it exists (2 mins)

Next time someone asks:
"Check the How We Do Things folder first - I recorded one
on that. If it doesn't answer your question, come back to me."

## That's it.

Do this every time someone asks you something.
In 3 months, you'll have documented half your business.
```

def calculate_customer_health_score(customer):
    """

    75+ = healthy
    """

    score = 0

    # ENGAGEMENT (40 points max)
    if customer.last_contact < 30_days:
        score += 20
    elif customer.last_contact < 60_days:
        score += 10

    if customer.responds_to_emails:
        score += 10

    if customer.attends_meetings:
        score += 10

    # VALUE RECEIVED (30 points max)
    if customer.using_all_services:
        score += 15
    elif customer.using_most_services:
        score += 10

    if customer.achieved_goals:
        score += 15

    # RELATIONSHIP (30 points max)
    if customer.nps_score >= 9:
        score += 15
    elif customer.nps_score >= 7:
        score += 10

    if customer.referred_others:
        score += 10

    if customer.expanding_spend:
        score += 5

    return {
