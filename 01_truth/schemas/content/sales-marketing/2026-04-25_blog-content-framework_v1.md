## Publishing Checklist

### Pre-Publish
- [ ] SEO title includes primary keyword
- [ ] Meta description 160 characters or less
- [ ] Primary keyword in first 100 words
- [ ] 2-3 secondary keywords used naturally
- [ ] H2/H3 headings use keyword variations
- [ ] Alt text includes keyword for images
- [ ] URL slug is SEO-friendly
- [ ] Word count 2,000-2,500 words
- [ ] Internal links (3-4 to other blog posts)
- [ ] External links (2-3 authoritative sources)
- [ ] Flesch Reading Ease 60+

### Content
- [ ] No corporate jargon
- [ ] Specific numbers used
- [ ] Conversational tone
- [ ] Validates reader's current approach
- [ ] Low-friction messaging
- [ ] No vague adjectives
- [ ] No salesy hype

### Conversion
- [ ] CTA visible above fold
- [ ] Lead magnet specific and valuable
- [ ] Book audit button present
- [ ] Email signup option
- [ ] Related posts section

### Technical
- [ ] Mobile responsive tested
- [ ] Page load under 3 seconds
- [ ] No broken links
- [ ] Images optimized
- [ ] Structured data markup

## Repurposing Workflow

### Blog Post → LinkedIn Post
1. Extract key statistic or insight
2. Write 500-character post with hook
3. Add CTA to read full post
4. Post with blog featured image

### Blog Post → Email Newsletter
1. Summarize main points in 300 words
2. Include 1-2 key statistics
3. Add full blog link
4. Include secondary CTA

### Blog Post → Short Video Script
1. Identify 3 key points
2. Write 3-5 minute script
3. Include hook and CTA
4. Film with phone

### Blog Post → Twitter Thread
1. Break into 5 tweets
2. Add hook tweet
3. Number each tweet
4. End with CTA

### Blog Post → Reddit Post
1. Share in relevant subreddit
2. Add value beyond blog
3. Engage with comments
4. Link to full post
