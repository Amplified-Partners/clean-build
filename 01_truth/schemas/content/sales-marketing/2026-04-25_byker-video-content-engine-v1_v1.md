# Byker Video & Content Production Engine

## Philosophy: Value First, Always

**Not:** "You have a problem you didn't know about, let me scare you"
**Instead:** "Here's a problem you're already feeling, here's how to fix it yourself"

This is the Ziglar/Godin/Cialdini approach done right:
- **Ziglar:** Help enough people get what they want, you'll get what you want
- **Godin:** Be so useful they can't ignore you
- **Cialdini:** Reciprocity through genuine value, not manipulation

## The Ethical Persuasion Framework

### Cialdini's 6 Principles - White Hat Application

| Principle | Dark Hat (DON'T) | White Hat (DO) |
|-----------|------------------|----------------|
| **Reciprocity** | Give teaser, demand email | Give full value freely, mention you help further |
| **Commitment** | Pressure small yeses to trap | Ask if they relate, let them self-identify |
| **Social Proof** | Fake testimonials, inflated numbers | Real case studies with permission |
| **Authority** | Fake credentials, borrowed expertise | Share your actual experience, be honest about limits |
| **Liking** | False rapport, over-familiarity | Genuine understanding of their world |
| **Scarcity** | Fake deadlines, false limits | Honest capacity constraints if real |

### The Value-First Content Formula

```
HOOK (3 seconds)
"Most plumbers lose £2,000/month on this one mistake"
↓
RELATE (10 seconds)
"You quote a job, they say yes, you do the work..."
"Then you wait 30, 60, 90 days for payment"
"Meanwhile, your suppliers want paying NOW"
↓
EDUCATE (30-45 seconds)
"Here's the fix: Change your payment terms TODAY"
"50% deposit on booking, 50% on completion"
"I'll show you exactly how to word it..."
[Actually show the template, the words, the script]
↓
RESULT (10 seconds)
"One of our clients did this last month"
"Went from £12k outstanding to £3k in 6 weeks"
↓
SOFT CTA (5 seconds)
"If you want the full template pack, link in bio"
"Or just screenshot this and do it yourself"
```

### What Makes It White Hat

1. **The advice works without you** - They can take the tip and never buy anything
2. **No artificial urgency** - "This offer expires!" when it doesn't
3. **Real numbers** - If you say "save £2k/month", show the maths
4. **Honest positioning** - "This works for service businesses under £500k"
5. **Respect intelligence** - Don't explain obvious things, don't patronise

## Video Production Pipeline (AI-Automated)

### The Byker Content Factory Flow

```python
class ContentFactory:
    """AI-controlled content production pipeline."""

    async def create_content_batch(self, pain_point: str, target: str):
        """
        From one pain point, generate full content suite.

        Args:
            pain_point: e.g., "cash_flow_timing"
            target: "owner" or "staff"
        """

        # Stage 1: Research & Script
        research = await self.llm_router.complete(
            task_type="content_creation",
            prompt=f"""
            Pain point: {pain_point}
            Target audience: {target}

            Research this pain point for UK SMBs:
            1. What specifically causes this pain?
            2. What have they probably already tried?
            3. What's the actual solution (be specific)?
            4. What's one thing they can do TODAY?

            Use UK terminology and examples.
            """,
            model="claude-sonnet"  # Quality matters for scripts
        )

        # Stage 2: Generate Scripts
        scripts = {
            "webinar": await self.generate_webinar_script(research, pain_point),
            "short_video": await self.generate_short_script(research, pain_point),
            "carousel": await self.generate_carousel_script(research, pain_point),
            "blog": await self.generate_blog_post(research, pain_point),
        }

        # Stage 3: Production Queue
        production_tasks = []

        # Short video via HeyGen API
        production_tasks.append(
            self.heygen.create_video(
                script=scripts["short_video"],
                avatar="ewan_avatar",  # Your cloned avatar
                voice="ewan_voice",    # Your cloned voice
                style="casual_professional"
            )
        )

        # Carousel via Canva API
        production_tasks.append(
            self.canva.create_carousel(
                slides=scripts["carousel"],
                template="byker_brand"
            )
        )

        # Blog post to Obsidian
        production_tasks.append(
            self.obsidian.create_note(
                path="30-CONTENT/drafts",
                content=scripts["blog"],
                tags=[pain_point, target]
            )
        )

        # Execute all in parallel
        results = await asyncio.gather(*production_tasks)

        return {
            "pain_point": pain_point,
            "scripts": scripts,
            "assets": results,
            "status": "ready_for_review"
        }
```

### HeyGen Integration

```python
class AvatarVideoService:
    """Generate AI avatar videos via HeyGen API."""

    def __init__(self):
        self.api_key = os.getenv("HEYGEN_API_KEY")
        self.avatar_id = "ewan_byker"  # Your trained avatar
        self.voice_id = "ewan_voice"   # Your cloned voice

    async def create_educational_video(
        self,
        script: str,
        style: str = "professional",
        background: str = "office",
        output_format: str = "vertical"  # For TikTok/Reels
    ):
        """
        Create a talking-head video from script.

        Returns video URL and local path.
        """
        response = await self.heygen_api.create_video(
            avatar_id=self.avatar_id,
            voice_id=self.voice_id,
            script=script,
            video_style=style,
            background=background,
            aspect_ratio="9:16" if output_format == "vertical" else "16:9"
        )

        # Wait for rendering
        video_url = await self.poll_completion(response["video_id"])

        # Download and store locally
        local_path = await self.download_video(
            video_url,
            f"30-CONTENT/videos/{datetime.now().isoformat()}.mp4"
        )

        return {
            "url": video_url,
            "local_path": local_path,
            "duration": response["duration"],
            "script": script
        }
```

## Webinar Production System

### The Value Webinar Formula

**Not:** "30 minutes of teasing, 5 minutes of pitch"
**Instead:** "55 minutes of pure value, 5 minutes of 'here's how we help further'"

```
WEBINAR STRUCTURE (60 minutes)

0:00 - 5:00   | INTRODUCTION
               "Here's exactly what you'll learn today..."
               [List 3-5 specific outcomes]
               "You'll be able to implement this yourself by the end"

5:00 - 15:00  | THE PROBLEM (Deep)
               Why this pain exists
               What most people try (and why it fails)
               The hidden cost of not fixing it
               [Real numbers, real examples]

15:00 - 45:00 | THE SOLUTION (Detailed)
               Step 1: [Specific action with example]
               Step 2: [Specific action with example]
               Step 3: [Specific action with example]

               [Screen share actual tools/templates]
               [Show real implementations]
               [Answer "but what about..." objections]

45:00 - 55:00 | IMPLEMENTATION
               "Here's your homework for this week"
               Specific actions with deadlines
               Resources/templates you're giving away
               Q&A on the material

55:00 - 60:00 | THE OFFER (Soft)
               "If you want to do this with guidance..."
               "Here's how we work with businesses like yours"
               "Book a diagnostic call, no obligation"
               [Clear next step, no pressure]
```

## The Ethical Bottom Line

**Test every piece of content against this:**

1. Could someone implement this without ever paying me?
2. Am I solving a real problem they already have?
3. Would I be proud to show this to my kids?
4. Does this make their life genuinely better?

If yes to all four, publish. If not, revise or bin it.

This is how you build trust at scale. This is how you become the obvious choice. This is white hat marketing that works.
